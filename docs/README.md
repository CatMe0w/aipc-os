# Documentation

This directory contains reverse-engineering notes and reference documents for the Anyka AK7802 SoC and AIPC netbook.

## Index

- [bootrom](bootrom/README.md): Reverse-engineered AK7802 SoC bootrom architecture, boot flow, boot sources, interfaces, and image format.
- [nboot](nboot/README.md): Reverse-engineered AIPC netbook firmware v1.58.2 and v1.88 first-stage bootloader architecture and boot flow.
- [eboot](eboot/README.md): Reverse-engineered second-stage bootloader WinCE EBOOT - memory map, GPIO/NAND/LCD/Ethernet drivers, partition format, download protocol, and maintenance mode.
